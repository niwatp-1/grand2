# THE GRAND — ระบบจัดการเลข v2

ระบบจัดการหวยแบบ Single-file HTML ใช้งานได้ทันทีโดยไม่ต้องติดตั้งอะไรเพิ่ม

---

## วิธีใช้งาน

ดาวน์โหลดไฟล์ `grand_standalone.html` แล้วเปิดด้วย browser ได้เลย ไม่ต้องมี internet ไม่ต้อง server

ข้อมูลทั้งหมดเก็บใน **localStorage** ของ browser นั้น — ถ้าล้าง browser data ข้อมูลจะหาย ควร Backup เป็นประจำ

---

## บัญชีเริ่มต้น

| Role | ID | รหัสผ่าน | หมายเหตุ |
|------|----|----------|---------|
| เจ้าของ | `Admin` | `1234` | — |
| หัวสาย | `LP88` | `1234` | คอม 20% |
| หัวสาย | `LP99` | `1234` | คอม 25% |
| คนคีย์ | `OP1` | `111` | สังกัด LP88 |

> ระบบ login จะ detect role อัตโนมัติจาก ID ที่กรอก ไม่ต้องเลือก role

## สิทธิ์แต่ละ Role

### เจ้าของ
- ดู dashboard ยอดขายรวม, กราฟ, ความเสี่ยง
- ดูรายการขายทั้งหมด + กรองตามประเภท / หัวสาย
- กรอกผลรางวัล + คำนวณผู้ถูกรางวัล
- ตั้งราคาจ่ายทุกประเภท
- จัดการหัวสาย (เพิ่ม / แก้ไข / ลบ)
- Export CSV / Backup JSON / Restore

### หัวสาย
- ดูยอดขายของตัวเอง แยกตามคนคีย์
- ดูรายการขาย + ค้นหา
- จัดการคนคีย์ในสังกัด (เพิ่ม / แก้ไข)

### คนคีย์
- รับออเดอร์ลูกค้า บันทึกเลข
- ดูรายการที่ตัวเองกรอก + ค้นหา + แก้ไข / ลบ

## ประเภทที่รองรับ

| ประเภท | หลัก | อัตราจ่าย (default) |
|--------|------|---------------------|
| 2 ตัวบน | 2 | 90 |
| 2 ตัวล่าง | 2 | 90 |
| 2 ตัวกลับ | 2 | 90 |
| 3 ตัวบน | 3 | 900 |
| 3 ตัวโต๊ด | 3 | 150 |
| 3 ตัวล่าง | 3 | 450 |
| 3 ตัวหน้า | 3 | 450 |
| 3 ตัวกลับ | 3 | 150 |
| **6 ตัวกลับ** | **3** | **1,000** |
| วิ่งบน | 1 | 3.2 |
| วิ่งล่าง | 1 | 4.2 |

### 6 ตัวกลับ คืออะไร
กรอกเลข 3 หลัก ระบบจะบันทึก **6 รายการอัตโนมัติ** (permutations ทั้งหมดของ 3 ตัวเลขนั้น)

ตัวอย่าง: กรอก `123` → บันทึก `123`, `132`, `213`, `231`, `312`, `321`

ถูกรางวัลเมื่อ: เลขใดเลขหนึ่งใน 6 แบบตรงกับผล **6 ตัวกลับ** ที่เจ้าของกรอก (กรอกแยกจาก 3 ตัวบน)

## การกรอกผลรางวัล (เจ้าของ)

ไปที่ **รางวัล** → **กรอกผลรางวัล**

| Field | ใช้กับประเภท |
|-------|-------------|
| รางวัลที่ 1 (6 หลัก) | ระบบ auto-fill 3 ตัวบน, 3 ตัวโต๊ด, 3 ตัวกลับ, 2 ตัวบน, 2 ตัวกลับ, วิ่งบน |
| 3 ตัวล่าง | กรอกเอง |
| 3 ตัวหน้า | กรอกเอง (รองรับหลายตัว) |
| 2 ตัวล่าง | กรอกเอง |
| วิ่งล่าง | กรอกเอง |
| **6 ตัวกลับ** | **กรอกเอง 3 หลัก (อิสระ ไม่อ้างอิงรางวัลที่ 1)** |

กด **คำนวณรางวัล** เพื่อดูรายการที่ถูกและยอดจ่ายทั้งหมด

## Backup & Restore

- **Backup**: Owner header → ปุ่ม Backup → ได้ไฟล์ `.json`
- **Restore**: Owner header → ปุ่ม Restore → เลือกไฟล์ `.json`
- **Export CSV**: Tab รายการ → Export Excel (ได้ไฟล์ `.csv` เปิดใน Excel/Google Sheets ได้)

> แนะนำ Backup ทุกงวด

---

## เทคนิค

- **Stack**: React 19 + TypeScript (pre-compiled) — ไม่มี build step
- **Storage**: localStorage เท่านั้น
- **ขนาดไฟล์**: ~585 KB (รวม React runtime)
- **ฟอนต์**: Sarabun + IBM Plex Mono (โหลดจาก Google Fonts เมื่อมี internet, fallback system font เมื่อ offline)
- **Browser**: Chrome, Firefox, Safari, Edge (รองรับทุกขนาดหน้าจอ)
